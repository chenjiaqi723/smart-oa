# SmartOA 企业后台管理系统需求说明

## 1. 项目简介

SmartOA 是一个基于 Spring Boot 开发的企业后台管理系统，主要用于模拟企业内部办公管理场景，实现员工、部门、角色、权限等基础业务管理功能。

本项目采用前后端分离架构，后端基于 Spring Boot 开发，数据库使用 MySQL，缓存使用 Redis，并使用 Git 进行版本管理。

---

## 2. 项目目标

完成一个具有实际业务场景的企业管理系统，熟悉企业级项目开发流程，包括需求分析、数据库设计、接口开发、测试、Bug 修复及版本管理。

---
